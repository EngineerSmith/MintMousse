# Card Subtitle
Subtitles can be used anywhere, but generally speaking they're to provide additional information not suited ot be part of the title, but important to know. Can check out [Bootstrap's docs for an example](https://getbootstrap.com/docs/5.3/components/card/#titles-text-and-links)

## Updates
`text` _string_ (**nil**)
:   The text of the subtitle, if none the entire element is hidden.

## Examples
```lua
{
  text = "I'm subtitle text",
}
```
