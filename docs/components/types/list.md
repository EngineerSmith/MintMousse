# List
A container which separates it's children. Can check out [Bootstrap's docs for an example](https://getbootstrap.com/docs/5.3/components/list-group/#basic-example).

## Updates
`isNumbered` _boolean_ (**false**)
:   If the list should number each child, an [example can be seen here](https://getbootstrap.com/docs/5.3/components/list-group/#numbered).

## Examples
```lua
{ }

{
  isNumbered = true,
}
```
